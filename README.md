# Student Grade Calculator

A simple Java console application that calculates a student's total marks, percentage, and grade.

## Features

- Enter student name
- Enter marks for 5 subjects
- Calculate total marks
- Calculate percentage
- Display grade

## Technologies Used

- Java
- Visual Studio Code

## Concepts Used

- Scanner
- Variables
- If-Else
- Arithmetic Operators

## How to Run

1. Open the project in VS Code.
2. Run `Main.java`.
3. Enter the student's name and marks.
4. View the calculated report.
